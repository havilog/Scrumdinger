# Scrumdinger

## 주요 목표

1. 뷰 <-> 뷰 모델 간의 데이터 바인딩
2. 상위 뷰 <-> 하위 뷰 간의 정보 전달
3. SwiftUI에서 뷰의 네비게이션 방식
    - NavigationLink, NavigationStack, NavigationDestination
    - Stack Based Navigation vs Tree Based Navigation
    - sheet, popover, fullscreenCover
    - Alerts, Dialogs
    - Deeplinking
4. 의존성 주입과 테스팅
